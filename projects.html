In the fall of 2014, my web app class was given a group assignment to redesign one of wheaton college's webpages. My group chose the study abroad page. While others worked on giving the page a new look, I added a simple visualization of the study abroad locations. Using leaflet.js and geoJSON, I added location markers to a map of the globle color coded by region with the option to click on them to get a pop-up with the name of the program at that location. While no longer hosted on the web, the code for this project can be found on github.


This presentation was given in the fall of 2013 for a DNA bioinformatics class. Using python, the genomes of six bacteria and one randomized genome were analyzed looking for certain motifs to determine the similarity of the bacteria. Five infectious bacteria were chosen and one everyday, harmless bacterium was chosen as the subjects to compare. Initially I expected greater similarity between the infectious bacteria, but the results showed otherwise. The dendrogram's produced using python showed similarities depended greatly on genus and little similarity based on ability to infect and cause disease.
